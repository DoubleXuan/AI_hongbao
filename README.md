# 春节红包活动实时看板

一个轻量网页 APP，用于聚合和展示微信、腾讯混元（含浑元关键词）、通义千问等 AI APP 红包活动相关资讯。

## 功能

- 实时聚合多个新闻 RSS 数据源
- 平台筛选（微信 / 混元 / 千问 / 其他 AI）
- 关键词搜索过滤
- 手动刷新 + 每 60 秒自动刷新
- 数据源异常时自动降级提示

## 运行方式

```bash
npm start
```

启动后访问：`http://localhost:3000`

## 技术说明

- 无第三方依赖，Node.js 原生 `http + fetch`
- 后端接口：`GET /api/events`
- 前端：原生 HTML/CSS/JS
